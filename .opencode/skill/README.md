# Skills Index

This directory holds project OpenCode skills.

Use:
- `project-context` for any repo task
- `python-nautilus` for Python, Nautilus, clipboard, filesystem, and desktop-integration work
- `packaging-installation` for install flow and packaging changes
- `verification` for choosing the right checks to run
- `review` for diff review and final pass checks

Related agent prompts live in `.opencode/agent/`.

Related command prompts live in `.opencode/command/`.

Reusable practices live in `.docs/project/software-development.md`, `.docs/project/verification.md`, and `.docs/project/packaging.md`.

Naming convention:
- Use short, hyphenated names
- Keep each skill focused on one workflow or concern
- Prefer project-specific names over generic ones
